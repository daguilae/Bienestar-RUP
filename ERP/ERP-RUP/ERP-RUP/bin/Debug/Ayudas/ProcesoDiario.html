Creación de Libros

Con esta opción podremos crear nuevos libros desde Cero, llenando los datos necesarios para crear el encabezado del libro


Modificar Libros

Para modificar libros deberemos dar doble Click sobre el registro a modificar.

Esta opción activará el panel de modificación de datos, para hacr permanentes estos cabios deberemos presionar Actualizar.


Eliminar Libros

Esta opción se activará al dar doble click sobre un registro, se considera eliminado al desmarcar la casilla de estado del panel de modificación.

Guardar Partidas

Para proceder a guardar las partidas contables en un libro, debemos presionar la opción de "Ver detalles"

Esto no llevará a la siguiente vista, en la cual procederemos a guardar partidas contables

Para guardar la partida deberemos llevar cierta información como la Fecha y concepto

Para Guardar las partidas deberemos, tomar en cuenta las siguientes validaciones:


Espacios Vacíos en la partida

Formato de datos:

Sumas Iguales de la partida:

Si se cumple con estas validaciones se guardará la partida exitosamente!


Eliminar Partidas

Para Eliminar un partida de un libro deberemos seleccionar la partida que deseamos eliminar

Luego deberemos presionar la opción "Eliminar"

Se consultará si la acción a realizar es correcta, de serlo al aceptar se procederá a eliminar la partida y todos sus movimientos


Vizualizar y Exportar Paritdas

Si selecionamos la Opcion de Vizualización, esto nos lleva a una vista completa del libro

Para exportar presionaremos el bóton PDF, lo que nos abrirá las opciones para guradar el libro

Al terminar este proceso, se creará un archivo en Formato PDF con todo el contenido del libro
