Creación de Libros

Con esta opción podremos crear nuevos libros desde Cero, llenando los datos necesarios para crear el encabezado del libro


Modificar Libros

Para modificar libros deberemos dar doble Click sobre el registro a modificar.

Podremos cambiar los datos en el panel de modificación.


Eliminar Libros

Se considera eliminado un libro al definir el estado como inactivo


Generar Detalles

Con esta opción podremos crear los detalles del libro, para esto presionaremos la opción ver detalles.

Esto nos llevará a la vista de generación de detalles

Si presionamos iniciará el proceso de generación del libro


Vizualizar y Exportar Libros

Con esta opción podremos Ver los libros completos y exportarlos en formato PDF

Para exportar presionaremos el botón PDF, lo que nos abrirá las opciones para guradar el libro

Al terminar este proceso, se creará un archivo en Formato PDF con todo el contenido del libro
