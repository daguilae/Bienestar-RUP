Conceptos de empleado


    Formulario de adignacion de conceptos

    En este formulario se agregarán los conceptos que se le quiere agregar a los empleados. Para ello se tiene que seleccionar el tipo de concepto en el combo box luego se selecciona si se suma o se resta, luego se ingresa el monto que se le va a asignar y por último si es un tiempo indefinido no se activa el checkbox de lo contrario si se quiere asignar fecha se activa y se selecciona la fecha deseada y luego se ingresa. Este proceso se repite hasta agregar todos los conceptos deseados.
